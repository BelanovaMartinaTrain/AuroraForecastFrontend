# Aurora forecast

Website to display data needed to forecast Northern lights

## Fetures

-   find many stats on one dashboard page
-   weather forecast
-   graphs
-   handy unit switchers
-   information modals
-   accessibilty
-   fast-loading page with full score on Lighthouse (desktop :))

## Getting Started

```bash
npm install
npm run build
npm start
```

## Tech stack

-   Typescript
-   Next.js 14.1
-   TailwindCSS
-   ChartJS
